# ClaudeCoder Directory

A community directory for Claude Code commands, MCP servers, and workflows.

**Live Site**: [claudecoder.directory](https://claudecoder.directory)

## About

ClaudeCoder Directory is a curated collection of:

- **Commands** - Slash commands and automation scripts for Claude Code
- **MCP Servers** - Model Context Protocol integrations
- **Workflows** - Visual automation pipelines

## Tech Stack

- Next.js 14 (App Router)
- PostgreSQL with pgvector
- Prisma ORM
- Tailwind CSS
- Deployed on Vercel
